# Linear Algebra for Graphics

This project is a theory-driven linear algebra learning project designed for computer graphics.

The goal is not only to learn matrix computation, but to understand how vectors, matrices, coordinate systems, transformations, projections, rotations, interpolation, and numerical linear algebra support the modern graphics pipeline.

## Project Goals

This project aims to connect:

- rigorous linear algebra definitions;
- concrete numerical examples;
- NumPy implementations;
- visualization;
- computer graphics applications.

## Core Topics

1. Vectors, points, and coordinate systems
2. Dot product, norms, angles, and projections
3. Cross product and surface normals
4. Matrices as linear transformations
5. Change of basis and coordinate transformations
6. Affine transformations and homogeneous coordinates
7. Rotations and quaternions
8. Camera, view matrix, and projection matrix
9. Barycentric coordinates and triangle interpolation
10. Lighting and shading
11. Eigenvalues, PCA, and geometry processing
12. SVD, least squares, and numerical stability

## Learning Style

Each chapter follows the same structure:

1. Graphics motivation
2. Mathematical definition
3. Concrete numerical example
4. Core proposition or theorem
5. Proof sketch
6. NumPy implementation
7. Visualization
8. Graphics application
9. Common mistakes
10. Exercises
